# DIOTECH Website

Static landing page for DIOTECH — Precision Surface Restoration.

This version is intentionally image-light. It uses CSS-generated industrial visuals so the site can be deployed immediately without relying on inaccurate placeholder photos. Replace the visual blocks with real DIOTECH field photos when available.

## Files

- `index.html` — page structure and copy
- `styles.css` — full visual styling
- `script.js` — quote form email-draft behavior

## Deployment

### GitHub Pages

1. Open the repo settings.
2. Go to **Pages**.
3. Set source to `Deploy from a branch`.
4. Select `main` and `/root`.
5. Save.

### Replit

1. Create a new HTML/CSS/JS Repl.
2. Upload `index.html`, `styles.css`, and `script.js`.
3. Press Run.

### Static hosting / Hostinger

Upload the three files to the public web root.

## Required edits before launch

- Replace `info@diotech.ca` in `script.js` if a different intake email is used.
- Replace CSS visual placeholders with real before/after photos once DIOTECH has accurate field imagery.
- Confirm insurance before using any insurance language publicly.
- Add privacy/terms pages before collecting personal information through a backend form.
- Keep HST, test-patch, and site-condition language clear in public pricing copy.

## Positioning

DIOTECH is positioned as mobile precision surface restoration, not generic cheap cleaning.

Core copy:

> Mobile laser cleaning and surface preparation for metal, wood, marine, fleet, and commercial assets worth preserving.
